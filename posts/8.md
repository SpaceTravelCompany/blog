---
title: 타이포그래피의 중요성
---

웹 디자인에서 타이포그래피가 사용자 경험에 미치는 영향과 올바른 폰트 선택 방법을 설명합니다.

## 타이포그래피 기초

### 폰트 분류
- **세리프**: Times New Roman, Georgia (전통적, 신뢰감)
- **산세리프**: Arial, Helvetica (현대적, 깔끔)
- **모노스페이스**: Consolas, Monaco (코드용)

### 폰트 페어링
대비를 통한 조화를 추구하세요. 제목에는 특징적인 폰트, 본문에는 가독성 높은 폰트를 사용합니다.

## 가독성 향상 팁

### 줄 높이 (Line Height)
본문 텍스트는 1.5~1.8 정도의 줄 높이가 적합합니다.

### 줄 길이
최적의 줄 길이는 45~75자 정도입니다.

### 글자 간격
기본 설정을 유지하되, 대문자 제목은 약간 넓히는 것이 좋습니다.

## 반응형 타이포그래피

```css
html {
  font-size: clamp(16px, 2vw, 20px);
}
```

## 웹 폰트 최적화

- `font-display: swap` 사용
- WOFF2 포맷 우선
- 필요한 글리프만 서브셋팅

좋은 타이포그래피는 콘텐츠의 가치를 높입니다.
