## 1. 밑이 2인 로그의 정의

**핵심 공식:**

$$log_2 N = k \iff 2^k = N$$

- **왼쪽:** "N의 밑이 2인 로그는 k다" → **몇 번 제곱했는지**가 답(k)
- **오른쪽:** "2를 k번 곱하면 N이 된다" → 지수 식
- **⟺:** 서로 같은 말(동치). 로그와 지수는 **역함수** 관계

**예:** \(log_2 8 = 3\) 인 이유는 \(2^3 = 8\) 이기 때문.

## 2. 자주 헷갈리는 것들

### log / ln / log₂

| 표기 | 의미 | 밑 |
|------|------|-----|
| \(\log\) (수학) | 보통 \(\log_{10}\) (상용로그) | 10 |
| \(\ln\) | 자연로그 | \(e\) (약 2.718…) |
| \(\log_2\) | 이진 로그 | 2 |

**프로그래밍:** `log()`는 보통 **자연로그(ln)**, \(\log_2\)는 `log2()` 또는 `Math.log2()` 등으로 따로 둠.

### "로그의 답"이 뭔지

- **로그의 값 = 지수**
- \(log_2 N = k\) 에서 **k는 "2를 몇 번 곱했는지"**
- 그래서 \(log_2 16 = 4\) → 2를 4번 곱하면 16

---

## 3. 자주 쓰는 공식

- **밑 변환:** \(\displaystyle \log_a b = \frac{\log_c b}{\log_c a}\) (c는 10, e 등 아무거나)
- **곱 → 합:** \(\log(AB) = \log A + \log B\)
- **나눗셈 → 뺄셈:** \(\log(A/B) = \log A - \log B\)
- **거듭제곱:** \(\log A^p = p \cdot \log A\)
- **밑과 진수가 같을 때:** \(\log_a a = 1\), \(\log_a 1 = 0\)

---

## 4. 알고리즘에서 \(\log N\) 말할 때

- **Big-O에서 \(\log N\)** → 보통 **밑 2** (\(log_2 N\)) 로 생각함 (이진 탐색, 이진 트리 높이 등)
- 밑이 바뀌어도 상수 배 차이만 나서, **O(log N)** 에서는 밑을 안 씀
