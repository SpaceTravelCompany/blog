효과적인 웹 디자인을 위한 색상 이론의 기초와 실제 적용 사례를 통해 색상의 힘을 이해합니다.

## 색상 이론 기초

### 색상 휠
- **원색**: 빨강, 파랑, 노랑
- **2차색**: 주황, 초록, 보라
- **3차색**: 원색과 2차색의 조합

### 색상 조화

1. **보색**: 색상 휠의 반대편 색상 (높은 대비)
2. **유사색**: 인접한 색상 (조화로움)
3. **삼각색**: 120도 간격의 세 색상

## 웹 디자인에서의 활용

### 색상 심리학
- **파랑**: 신뢰, 전문성
- **초록**: 성장, 자연
- **빨강**: 긴급함, 열정
- **노랑**: 낙관, 따뜻함

### 접근성 고려
WCAG 가이드라인에 따라 최소 4.5:1의 명암비를 유지하세요.

## CSS 변수 활용

```css
:root {
  --primary: #3498db;
  --secondary: #2ecc71;
  --accent: #e74c3c;
}
```

일관된 색상 시스템은 브랜드 인지도를 높이고 사용자 경험을 개선합니다.
