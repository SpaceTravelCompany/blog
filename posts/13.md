### 복합 타입

```se

[10]T        // 고정 배열
[?]T         // 길이 미정
[dynamic]T   // 내장 동적 배열

*T, mut *T, unsafe *T : 포인터, 쓰기 전용 포인터, 안전하지 않은(C 라이브러리 연동 용) 포인터,

- `*T`는 동시에 여러 읽기 접근을 허용한다.
- `*mut T`는 동시에 하나의 쓰기 접근만 허용한다.
- 읽기 접근과 쓰기 접근은 동시에 공존할 수 없다.

```

## 10. 할당 규칙(Allocator)

원래는 GC 구간 noGC 구간을 나눌려고 했는데 그렇게 하지 않고,
GC를 쓰려면 메모리 자동으로 해제해주는 GC 전용 allocator 사용하고,
안쓰면 일반 Allocator 나 사용자 지정 Allocator를 사용한다.
\#freeAll(allocator) 함수로 GC allocator를 전달해서 강제 GC 발동 또는, 사용자 지정 arena allocator를 해제할수 있다.

\#allocator 예약어 변수를 수정해서 allocator를 수정할 수 있는데 #allocator 은 글로벌 범위가 아니라 context 규칙 범위이다.
즉, A 함수 내에서 수정하면 A 함수 내의 다른 함수 내에도 적용되고, A 함수 밖에는 적용 안되는 식이다.

new, delete 를 사용하면 ~~구조체 내에 정의된 생성자, 소멸자가 호출된다.~~ 값이 자주 복사 되기 때문에 (숨겨진 제어 흐름이기도 하고) 없는게 좋겠다.
