최신 JavaScript 표준인 ES2024에서 추가된 새로운 기능들을 살펴보고, 실제 프로젝트에 어떻게 활용할 수 있는지 알아봅니다.

## 주요 신규 기능

### 1. Array Grouping
배열을 특정 조건으로 그룹화하는 `Object.groupBy()`와 `Map.groupBy()` 메서드가 추가되었습니다.

```javascript
const items = [
  { name: 'apple', type: 'fruit' },
  { name: 'carrot', type: 'vegetable' }
];

const grouped = Object.groupBy(items, item => item.type);
```

### 2. Promise.withResolvers()
Promise의 resolve, reject 함수를 외부에서 접근할 수 있게 해줍니다.

```javascript
const { promise, resolve, reject } = Promise.withResolvers();
```

### 3. Well-formed Unicode Strings
유니코드 문자열 처리가 더욱 안정적으로 개선되었습니다.

## 브라우저 지원

최신 Chrome, Firefox, Safari에서 대부분의 기능을 지원합니다. 프로덕션에서는 Babel 등의 트랜스파일러 사용을 권장합니다.
