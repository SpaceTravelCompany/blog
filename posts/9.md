## 1. glslang이란

**glslang**[https://github.com/KhronosGroup/glslang]은 Khronos의 공식 GLSL/HLSL 프론트엔드로, 셰이더 소스 코드를 **SPIR-V**로 컴파일한다. Vulkan은 SPIR-V 바이트코드를 사용하므로, 런타임에 GLSL 문자열을 SPIR-V로 변환하려면 glslang(또는 shaderc 등)이 필요하다.

---

## 2 glslang 프로세스 초기화/종료

glslang은 **프로세스 단위**로 초기화한다(스레드 아님). 한 번만 호출하고, 종료 시 `glslang_finalize_process`를 호출한다.

```c
#include <glslang/Include/glslang_c_interface.h>

/* At program startup */
if (glslang_initialize_process() == 0) {
    /* failure */
}

/* At shutdown */
glslang_finalize_process();
```

멀티스레드 환경에서는 한 번만 초기화되도록 한다.

## 3. 컴파일 흐름

### 3.1 입력 구조체 (`glslang_input_t`)

```c
#include <stdbool.h>
#include <stdint.h>
#include <glslang/Include/glslang_c_interface.h>
#include <glslang/Public/resource_limits_c.h>

glslang_input_t input = {
    .language = hlsl ? GLSLANG_SOURCE_HLSL : GLSLANG_SOURCE_GLSL,
    .stage = stage, /* e.g. GLSLANG_STAGE_VERTEX, GLSLANG_STAGE_FRAGMENT, ... */
    .client = GLSLANG_CLIENT_VULKAN,
    .client_version = GLSLANG_TARGET_VULKAN_1_4,
    .target_language = GLSLANG_TARGET_SPV,
    .target_language_version = GLSLANG_TARGET_SPV_1_6,
    .code = code_str, /* shader source (NUL-terminated) */
    .default_version = 100,
    .default_profile = GLSLANG_NO_PROFILE,
    .force_default_version_and_profile = 0,
    .forward_compatible = 0,
    .messages = GLSLANG_MSG_DEFAULT_BIT,
    .resource = glslang_default_resource(),
    .callbacks = { 0 },
    .callbacks_ctx = NULL,
};
```

인클루드 경로는 빌드·패키지마다 다를 수 있다. 위 이름은 Khronos [glslang C 인터페이스](https://github.com/KhronosGroup/glslang/blob/main/glslang/Include/glslang_c_interface.h) 기준이다.

### 3.2 Vulkan 버전에 따른 SPIR-V 버전

Vulkan 버전에 맞게 `target_language_version`과 `client_version`을 조정한다. 근데 버전을 올린다고 성능이 좋아지는건 아니라고 한다. 그냥 1.0, 1.0으로 호환성을 유지해도 되겠다.(glslc를 실행했을때 기본값이기도하고)


| Vulkan | `client_version` | `target_language_version` |
| ------ | ---------------- | --------------------------- |
| 1.0    | `GLSLANG_TARGET_VULKAN_1_0` | `GLSLANG_TARGET_SPV_1_0` |
| 1.1    | `GLSLANG_TARGET_VULKAN_1_1` | `GLSLANG_TARGET_SPV_1_3` |
| 1.2    | `GLSLANG_TARGET_VULKAN_1_2` | `GLSLANG_TARGET_SPV_1_5` |
| 1.3    | `GLSLANG_TARGET_VULKAN_1_3` | `GLSLANG_TARGET_SPV_1_6` |
| 1.4+   | `GLSLANG_TARGET_VULKAN_1_4` | `GLSLANG_TARGET_SPV_1_6` |

## 4. 컴파일 단계

### 4.1 셰이더 생성 및 전처리

```c
#include <stdbool.h>
#include <stdint.h>

glslang_shader_t *shader = glslang_shader_create(&input);
if (shader == NULL) {
    /* allocation failed */
}

glslang_shader_set_entry_point(shader, "main");

if (glslang_shader_preprocess(shader, &input) == 0) {
    const char *info_log = glslang_shader_get_info_log(shader);
    const char *debug_log = glslang_shader_get_info_debug_log(shader);
    (void)info_log;
    (void)debug_log;
    glslang_shader_delete(shader);
    return false; /* or your error path */
}
/* ... later: glslang_shader_delete(shader); on success path too */
```

### 4.2 파싱

```c
if (glslang_shader_parse(shader, &input) == 0) {
    /* check glslang_shader_get_info_log / get_info_debug_log */
    glslang_shader_delete(shader);
    return false;
}
```

### 4.3 프로그램 생성 및 링크

```c
glslang_program_t *program = glslang_program_create();
if (program == NULL) {
    glslang_shader_delete(shader);
    return false;
}

glslang_program_add_shader(program, shader);

int link_messages = (int)GLSLANG_MSG_SPV_RULES_BIT | (int)GLSLANG_MSG_VULKAN_RULES_BIT;
if (glslang_program_link(program, link_messages) == 0) {
    /* link failed: glslang_program_get_info_log etc. */
    glslang_program_delete(program);
    glslang_shader_delete(shader);
    return false;
}
/* ... glslang_program_delete(program); when done */
```

### 4.4 SPIR-V 생성

```c
#include <stdlib.h>
#include <stdint.h>

glslang_spv_options_t spv_options = {
    .generate_debug_info = false,
    .strip_debug_info = false,
    .disable_optimizer = false,
    .optimize_size = false,
    .disassemble = false,
    .validate = true,
    .emit_nonsemantic_shader_debug_info = false,
    .emit_nonsemantic_shader_debug_source = false,
    .compile_only = false,
    .optimize_allow_expanded_id_bound = false,
};

#ifndef NDEBUG
spv_options.generate_debug_info = true;
spv_options.disable_optimizer = true;
spv_options.emit_nonsemantic_shader_debug_info = true;
spv_options.emit_nonsemantic_shader_debug_source = true;
#endif

glslang_program_SPIRV_generate_with_options(program, stage, &spv_options);

size_t spirv_word_count = glslang_program_SPIRV_get_size(program);
uint32_t *spirv_words = malloc(sizeof(uint32_t) * spirv_word_count);
if (spirv_words == NULL) {
    /* out of memory */
}
glslang_program_SPIRV_get(program, spirv_words);
```

## 5. Vulkan ShaderModule 생성

glslang으로 얻은 SPIR-V 바이트코드를 Vulkan **ShaderModule**으로 등록한다.

### 5.1 vkCreateShaderModule

```c
VkShaderModuleCreateInfo createInfo = {
    .sType    = VK_STRUCTURE_TYPE_SHADER_MODULE_CREATE_INFO,
    .codeSize = spirv_byte_count,      // SPIR-V 바이트 수 (4의 배수)
    .pCode    = spirv_data,            // SPIR-V uint32 배열
};

VkShaderModule shaderModule;
vkCreateShaderModule(device, &createInfo, NULL, &shaderModule);
```

- `codeSize`: SPIR-V 바이트 길이. SPIR-V는 4바이트 단위이므로 `sizeof(uint32_t) * word_count`.
- `pCode`: SPIR-V 바이트를 `uint32_t*`로 전달. `glslang_program_SPIRV_get`으로 채운 버퍼를 그대로 쓸 수 있다.

### 5.2 파이프라인에 사용

`VkPipelineShaderStageCreateInfo`에서 `module`에 이 ShaderModule을 지정하고, `pName`에 엔트리 포인트(예: `"main"`)를 넣는다.

```c
VkPipelineShaderStageCreateInfo stageInfo = {
    .sType  = VK_STRUCTURE_TYPE_PIPELINE_SHADER_STAGE_CREATE_INFO,
    .stage  = VK_SHADER_STAGE_VERTEX_BIT,  // 또는 FRAGMENT, COMPUTE 등
    .module = shaderModule,
    .pName  = "main",
};
```

### 5.3 정리

사용이 끝나면 `vkDestroyShaderModule(device, shaderModule, NULL)`로 해제한다.

## 6. 에러 처리

glslang 각 단계(전처리, 파싱, 링크) 실패 시:

```c
const char *info_log = glslang_shader_get_info_log(shader);
const char *debug_log = glslang_shader_get_info_debug_log(shader);
/* or: glslang_program_get_info_log(program), glslang_program_get_info_debug_log(program) */
/* log and treat as failure */
```

SPIR-V 생성 후 메시지가 있으면:

```c
#include <stdio.h>

const char *spirv_messages = glslang_program_SPIRV_get_messages(program);
if (spirv_messages != NULL && spirv_messages[0] != '\0') {
    fprintf(stderr, "%s\n", spirv_messages);
}
```
